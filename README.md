# Aspire Gym — Database Design & SQL Implementation

A fully designed and implemented relational database system 
for a gym management platform covering membership management, 
class scheduling, facility reservations and payment processing.

## Tech Stack
- SQL
- MySQL
- phpMyAdmin
- Visual Paradigm
- UML Design

## Database Tables
- Member — member profiles and health information
- Membership — subscription plans and status
- Staff — instructors and facility managers
- Facility — gym facilities and capacity
- Facility Booking — facility reservation management
- Payment — payment processing and transactions
- Class — fitness class scheduling
- Class Booking — class enrollment management

## Database Design
- Entity Relationship Diagram (ERD)
- Normalisation — 1NF, 2NF and 3NF applied
- Data Dictionary with full column descriptions
- Foreign key constraints and relationships
- Indexing for query performance

## SQL Implementation
- CREATE TABLE statements with primary and foreign keys
- INSERT statements with sample data
- Complex SELECT queries for reporting and analytics
